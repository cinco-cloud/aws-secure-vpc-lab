🚀 AWS Secure VPC Lab
📌 Overview

This project demonstrates the deployment of a secure AWS Virtual Private Cloud (VPC) environment with public and private subnets, routing, and access controls.

🧱 Architecture
1 VPC
Public Subnet (for web access)
Private Subnet (for internal resources)
Internet Gateway
Route Tables
Security Groups
EC2 Instance
🔐 Security Features
Restricted inbound traffic via Security Groups
Separation of public and private resources
Principle of least privilege applied
⚙️ Services Used
Amazon VPC
Amazon EC2
AWS IAM
🎯 Outcome

Built and configured a secure cloud network environment following AWS best practices.
